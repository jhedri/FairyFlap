Manual review steps (~2 minutes)

  1. Launch — Open Fairy Flap. Home screen shows title, scores, and PLAY.
  2. Start — Tap anywhere (or PLAY).
  3. Play — Tap to flap upward; avoid stone obstacles.
  4. Score — Pass through gaps to increase score.
  5. Game over — Hit an obstacle or ground; screen flashes, then returns home.
  6. Restart — Tap anywhere on home to play again.

No permissions, sign-in, or internet connection needed. Enable airplane mode to confirm offline play.

Single simulator:

  -destination 'platform=iOS Simulator,name=iPhone 17 Pro' \
  -only-testing:FairyFlapUITests

Small + standard + large simulators:

Pre-submission checklist

  • Release build succeeds
  • ./scripts/run-ui-tests.sh passes
  • Manual test on physical iPhone (launch → play → game over → restart)
  • Portrait gameplay confirmed
  • Offline / airplane mode — no crash on launch
  • Version and build number updated
  • Review video current (review.html)
